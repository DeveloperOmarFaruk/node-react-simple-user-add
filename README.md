## Project Details
#
Simple user information add has been worked on this repository. Using the data get method from backend to show in UI and using data post method from user to send backend data and show new data in UI.
#

#
1. Node JS
2. Express JS
3. Nodemon
4. API
5. React JS
6. React Hooks
7. HTML5
8. CSS3
9. Bootstrap5
